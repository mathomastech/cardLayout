Anatomy of a card

The card design has 4 main parts. They are the card (basic, left, right, small, medium, large), the header (left, right, center), the content and the footer (left, right center). Each card should use only what it needs, and nothing extra.The next few cards will show you the difference between left and right, and the different sizes.

Basic with Right Header

This is an example of the basic card. This card resizes itself vertically to match the amount of content contained within. The width of the card will span the length of the screen. You can use this by implementing the "cardBasic" class. This card is also using the left oriented header which is implemented by using the "cardHeaderLeft" class.

Small left card

This is an example of a card that encompasses a small left portion of the screen. The card is best used if you have only a small amount of information to display and you would like to display additional content beside this. Use this by implementing the "cardLeftSmall" class.

Small right card

This card is identical to the small left card, except that it occupy's the right side of the page. Use in the same fashion as the small left card, and implement using the "cardRightSmall" class.

Medium left card

This is an example of a card that encompasses a medium left portion of the screen. The card is best used if you have only a medium amount of information to display and you would like to display additional content beside this. Use this by implementing the "cardLeftMedium" class.

Medium right card

This card is identical to the medium left card, except that it occupy's the right side of the page. Use in the same fashion as the medium left card, and implement using the "cardRightMedium" class. This class is also using a right header by implementing the "cardHalfHeaderRight" class.!

Large left card
